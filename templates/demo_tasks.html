Features

  • ✅

    Taskiq Task Queue

    Redis-backed with automatic InMemory fallback

  • ✅

    Cron Scheduling

    @app.schedule("0 * * * *") for periodic tasks

  • ✅

    Exponential Backoff

    Automatic retry with delays: 1s, 2s, 4s, 8s, 16s

  • ✅

    Result Storage

    Track task progress and results with TTL (7 days)

  • ✅

    Dead-Letter Queue

    Automatic dead-letter handling for failed tasks

Code Example

@app.task()
async def send_email(to: str, subject: str):
    # Background task
    await email_service.send(to, subject)

@app.schedule("0 */6 * * *")  # Every 6 hours
async def cleanup_expired_tokens():
    await PasswordResetToken.delete_expired()

# Enqueue task
task_id = await send_email.kiq("user@example.com", "Hello")

# Check status
result = await app.broker.task_result_backend.get_result(task_id)
print(result.status)  # pending, running, success, failed
