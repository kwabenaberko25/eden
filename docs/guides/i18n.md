# Internationalization (i18n) 🌍

Eden provides native support for multi-lingual applications using the standard `gettext` industry format.

## Configuration

Set your supported locales and default language in your `.env` or app configuration.

```text
SUPPORTED_LOCALES=en,fr,es,de
DEFAULT_LOCALE=en
```

## The Middleware

Enable the locale middleware to automatically detect the user's preferred language via the `Accept-Language` header or a session key.

```python
app.add_middleware("i18n")
```

---

## Translating Content

### In Python Code

Use the `_()` function (alias for `gettext`) to mark strings for translation.

```python
from eden.i18n import _

async def welcome(request):
    message = _("Welcome to our garden!")
    return {"message": message}
```

### In Templates

In templates, use the standard double-curly braces for translations.

```html
<h1>{{ _("Welcome to our garden!") }}</h1>

{{ _("You have {count} message.").format(count=n) }}
```

### Pluralization

Eden supports pluralization via `ngettext`.

```html
{{ ngettext("You have {count} message.", "You have {count} messages.", n).format(count=n) }}
```

### Formatting Filters

| Filter | Description | Example |
| :--- | :--- | :--- |
| `date` | Locale-aware date. | `{{ task.due_at\|date }}` |
| `time` | Locale-aware time. | `{{ task.due_at\|time }}` |
| `number` | Thousand separators. | `{{ 1500000\|number }}` |
| `currency` | Localized symbol. | `{{ 50\|currency }}` |

## Managing Catalogs

Eden looks for translation files in your project's `locale/` directory.

```text
my_project/
└── locale/
    ├── fr/
    │   └── LC_MESSAGES/
    │       └── messages.po
    └── es/
        └── LC_MESSAGES/
            └── messages.po
```

### Compiling Translations

Use the Eden CLI to extract and compile your translation strings.

```bash
# Extract strings to .pot file
eden i18n extract

# Compile .po files to binary .mo files
eden i18n compile
```

---

## Best Practices

- **Placeholders**: Always use named placeholders (e.g., `{name}`) instead of positional ones for better translator context.
- **Lazy Loading**: Eden loads translations only when needed, ensuring your app remains fast regardless of the number of supported languages.
