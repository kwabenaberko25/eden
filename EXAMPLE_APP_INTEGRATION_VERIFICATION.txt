🧪 TESTING FEATURES

HTMX Testing:
  # Fragment request with HX-Target header
  curl -H "HX-Request: true" -H "HX-Target: items-list" http://localhost:8001/api/items

WebSocket Testing:
  # Open browser console and test via demo page, or:
  npm install -g wscat
  wscat -c ws://localhost:8001/ws

Background Tasks Testing:
  curl -X POST http://localhost:8001/api/send-email \
    -H "Content-Type: application/json" \
    -d '{"ticket_id": "123", "message": "Test"}'

Stripe Testing:
  curl -X POST http://localhost:8001/api/checkout \
    -H "Content-Type: application/json" \
    -d '{"plan": "pro"}'

Multi-Tenant Testing:
  # With tenant header
  curl -H "X-Tenant-ID: acme" http://localhost:8001/api/tenant-info
  
  # Without tenant header (allowed, since enforce=False)
  curl http://localhost:8001/api/tenant-info
